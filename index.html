Housing Data Dashboard

An interactive data visualization tool for housing market analysis

View on GitHub

Project Overview

This dashboard provides interactive visualizations for housing market data, allowing users to explore property prices, features, and market trends through an intuitive interface.

Note: This is a Python-based Dash application which requires a server to run. This page only hosts the documentation and code. To use the actual dashboard, please follow the installation instructions below.

Key Features

Interactive Mapping

Visualize property locations and prices on an interactive map to identify geographic patterns.

Price Distribution

Analyze housing price distributions across different property types and neighborhoods.

Market Trends

Track market trends and price changes over time with interactive time series visualizations.

Installation Guide

To run this dashboard locally, follow these steps:

1. Clone the repository
git clone https://github.com/Danik911/data-visualization-assignment.git
cd data-visualization-assignment
2. Set up a virtual environment
# Windows
python -m venv .venv
.venv\Scripts\activate

# macOS/Linux
python -m venv .venv
source .venv/bin/activate
3. Install dependencies
pip install -r requirements.txt
4. Configure environment variables

Create a .env file in the root directory with:

GOOGLE_MAPS_API_KEY=your_google_maps_api_key
5. Run the dashboard
python run_dashboard.py

The dashboard will be available at http://localhost:8050

Deployment Options

While GitHub Pages hosts this code repository, to deploy the live dashboard you can use:

  • Render
